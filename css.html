CSS: Cascading Style Sheets

CSS (Cascading Style Sheets u Hojas de Estilo en Cascada) es el lenguaje que determina cómo se ven los documentos HTML en pantalla. Mientras que HTML estructura el contenido de forma lógica, CSS controla el diseño visual: colores, tipografías, distribución espacial y la adaptabilidad a diferentes tamaños de dispositivos (diseño responsivo).

Se llama "en cascada" porque el navegador lee y aplica las reglas de diseño de arriba hacia abajo. Cuando hay conflictos (por ejemplo, dos reglas intentan cambiar el color del mismo texto), el motor de CSS decide cuál aplicar basándose en la "especificidad" (qué tan específica es la regla) y en la herencia de estilos de elementos padres a hijos.
